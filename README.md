# NumPy Showcase (PyQt6)

A simple desktop app that demonstrates core NumPy strengths:

- Vectorization speed vs Python loops
- Broadcasting for pairwise distances
- Linear algebra (`solve`, eigenvalues)
- Random simulation (Monte Carlo + random walk)

## Run

1. Install dependencies:
   ```powershell
   .\.venv\Scripts\python.exe -m pip install -r requirements.txt
   ```
2. Start the app:
   ```powershell
   .\.venv\Scripts\python.exe main.py
   ```
